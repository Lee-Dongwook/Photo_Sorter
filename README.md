# Photo Sorter

얼굴 인식 기반으로 사진을 인물별로 자동 분류하는 프로젝트입니다.

---

## 1. 기술 스택 및 라이브러리

| 구분                 | 기술                                            |
| -------------------- | ----------------------------------------------- |
| **Language**         | Python 3.9+                                     |
| **Core**             | `face-recognition` (dlib 기반)                  |
| **Image Processing** | Pillow (PIL) 또는 OpenCV                        |
| **GUI (선택)**       | Streamlit — 웹 대시보드 형태로 빠르게 구현 가능 |

---

## 2. 논리적 워크플로우 (System Architecture)

전체 프로세스는 다음 순서로 진행됩니다.

```
[학습 데이터 로드] → [대상 사진 스캔] → [얼굴 인코딩 비교] → [파일 분류]
```

### Reference (기준 사진)

- `kids/철수.jpg`, `kids/영희.jpg`처럼 **아이 이름**으로 된 폴더/파일명에서 인코딩 추출

### Target (분류 대상)

- 당일 촬영한 수백 장의 사진이 들어 있는 폴더

### Process

1. **Face Detection** — 대상 사진에서 얼굴 검출
2. **Distance** — 검출된 얼굴을 기준 사진들의 인코딩 값과 **유클리드 거리** 계산
3. **Matching** — 설정된 임계값(Threshold, 보통 0.6) 이하 중 가장 가까운 인물로 매칭

### Output

- `output/철수/`, `output/영희/` 폴더로 사진 **복사** 또는 **이동**
